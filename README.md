# HUSL-GLSL

HUSL is a human-friendly alternative to HSL. ( http://www.husl-colors.org )
GLSL port by William Malo ( https://github.com/williammalo )
Put this code in your fragment shader.

## api

	  xyzToRgb   (vec3 xyz   )
	  rgbToXyz   (vec3 rgb   )
	  xyzToLuv   (vec3 xyz   )
	  luvToXyz   (vec3 luv   )
	  luvToLch   (vec3 luv   )
	  lchToLuv   (vec3 lch   )
	 huslToLch   (vec3 husl  )
	  lchToHusl  (vec3 husl  )
	huslpToLch   (vec3 huslp )
	  lchToHuslp (vec3 lch   )
	  lchToRgb   (vec3 lch   )
	  rgbToLch   (vec3 rgb   )
	 huslToRgb   (vec3 husl  )
	  rgbToHusl  (vec3 rgb   )

	  luvToRgb   (vec3 luv   )
